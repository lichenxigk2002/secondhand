# 先看这里：项目交付入口

如果你是第一次接手这个项目，或者电脑上还没有安装开发软件，请按下面顺序阅读：

1. `docs/交付文档/00-新电脑软件安装清单.md`
2. `docs/交付文档/01-项目交付说明.md`
3. `docs/交付文档/02-演示流程与验收清单.md`

这三份文档分别解决：

- 新电脑需要安装哪些软件
- 项目怎么配置、启动、打开
- 演示和验收时按什么流程操作

如果只是答辩或演示，优先看第 2、3 份文档。

如果要写论文或整理材料，再看：

- `docs/毕设文档/毕业论文-正文初稿.md`
- `docs/指导文档/05-毕业论文写作指南.md`
- `docs/毕设文档/论文插图-Mermaid终稿.md`

注意：真实数据库密码、微信 AppSecret、AI API Key 不要写进公开文档，也不要提交到代码仓库。
