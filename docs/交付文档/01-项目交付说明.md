# 项目交付说明

本文档给不懂技术的同学使用。先按本文档跑通项目，再去看其他技术文档。

## 一、这个项目是什么

本项目是“邻物集”校园二手交易微信小程序，包含三个部分：

| 部分 | 目录 | 作用 |
|---|---|---|
| 微信小程序 | `miniapp/` | 给学生使用，发布商品、浏览附近商品、聊天、下单、评价 |
| 后端服务 | `backend/` | 提供登录、商品、图片、聊天、订单、评价等接口 |
| 后台管理系统 | `admin-web/` | 给管理员使用，审核商品、管理用户、处理举报、查看数据 |

## 二、交付内容

交付时至少应包含：

- 项目源代码：`miniapp/`、`backend/`、`admin-web/`
- 数据库迁移脚本：`backend/migrations/`
- 项目文档：`docs/`
- 环境变量示例：`backend/.env.example`、`miniapp/.env.example`、`admin-web/.env.example`
- 毕设材料：`docs/毕设文档/`

不要把真实密码、真实密钥、个人微信 AppSecret 发给别人。

## 三、第一次启动总流程

第一次运行时按这个顺序来：

1. 安装软件，见 `docs/交付文档/00-新电脑软件安装清单.md`
2. 配置后端 `.env`
3. 创建 MySQL 数据库
4. 启动后端 HTTP 服务
5. 启动 WebSocket 聊天服务
6. 启动小程序编译
7. 用微信开发者工具打开 `miniapp/dist`
8. 启动后台管理系统
9. 按演示流程验收功能

## 四、后端启动

打开 PowerShell，进入项目后端目录：

```bash
cd backend
```

创建 Python 虚拟环境：

```bash
py -m venv venv
```

说明：如果 `py` 命令不可用，也可以改用 `python -m venv venv`。

安装后端依赖：

```bash
.\venv\Scripts\python.exe -m pip install -r requirements.txt
```

如果下载很慢，可以使用国内镜像：

```bash
.\venv\Scripts\python.exe -m pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

后续后端命令都建议直接使用 `.\venv\Scripts\python.exe`，不需要手动激活虚拟环境。

复制配置文件：

```bash
copy .env.example .env
```

打开 `backend/.env`，至少修改下面几项：

```env
DB_USER=root
DB_PASSWORD=你的MySQL密码
DB_NAME=lbs_secondhand
WX_APPID=你的小程序AppID
WX_SECRET=你的小程序AppSecret
```

如果暂时不用 AI 辅助发布，可以先改成：

```env
AI_ENABLED=0
ARK_API_KEY=
```

创建数据库：

```bash
mysql -u root -p -e "CREATE DATABASE IF NOT EXISTS lbs_secondhand DEFAULT CHARACTER SET utf8mb4;"
```

执行数据库迁移：

```bash
.\venv\Scripts\python.exe -m flask db upgrade
```

启动后端 HTTP 服务：

```bash
.\venv\Scripts\python.exe run.py
```

看到服务运行在 `5002` 端口，表示后端 HTTP 服务启动成功。

## 五、聊天服务启动

再打开一个新的 PowerShell 窗口：

```bash
cd backend
.\venv\Scripts\python.exe ws_server.py
```

看到服务运行在 `5001` 端口，表示 WebSocket 聊天服务启动成功。

## 六、小程序启动

再打开一个新的 PowerShell 窗口：

```bash
cd miniapp
npm install
node scripts/create-ui-icons.js
npm run dev:weapp
```

编译成功后，用微信开发者工具打开：

```text
miniapp/dist
```

开发阶段需要在微信开发者工具里勾选：

```text
详情 -> 本地设置 -> 不校验合法域名、web-view、TLS 版本以及 HTTPS 证书
```

## 七、后台管理系统启动

再打开一个新的 PowerShell 窗口：

```bash
cd admin-web
npm install
copy .env.example .env
npm run dev
```

浏览器访问：

```text
http://127.0.0.1:5174
```

默认管理员账号：

```text
账号：admin
密码：admin123456
```

如果管理员表为空，首次登录接口会自动初始化默认管理员。

## 八、本机 IP 配置

小程序在手机或微信开发者工具里不能稳定使用 `localhost` 访问后端，建议使用电脑局域网 IP。

查看电脑 IP：

```bash
ipconfig
```

找到 `IPv4 地址`，例如：

```text
10.10.31.129
```

然后修改 `miniapp/.env`：

```env
TARO_APP_API=http://10.10.31.129:5002
TARO_APP_WS=ws://10.10.31.129:5001
```

修改后重新执行：

```bash
npm run dev:weapp
```

## 九、交付前检查

交付前确认：

- 后端 `.\venv\Scripts\python.exe run.py` 可以启动
- 聊天服务 `.\venv\Scripts\python.exe ws_server.py` 可以启动
- 小程序 `npm run dev:weapp` 可以编译
- 微信开发者工具能打开 `miniapp/dist`
- 后台 `npm run dev` 可以启动
- 管理员账号能登录
- 商品可以发布、浏览、聊天、完成订单、评价
- `backend/.env` 里真实密码和密钥没有发给不该拿到的人
