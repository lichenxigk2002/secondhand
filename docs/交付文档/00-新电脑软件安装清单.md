# 新电脑软件安装清单

本文档给第一次接手项目的同学使用。目标是把一台什么都没装的 Windows 电脑配置到可以运行本项目。

## 一、必须安装的软件

按下面顺序安装，全部安装完成后再启动项目。

| 顺序 | 软件 | 用途 | 建议版本 |
|---|---|---|---|
| 1 | 微信开发者工具 | 打开、预览、上传微信小程序 | 最新稳定版 |
| 2 | Node.js | 运行小程序前端和后台管理前端 | 18 LTS 或 20 LTS |
| 3 | Python | 运行 Flask 后端 | 3.10 或 3.11 |
| 4 | MySQL | 保存用户、商品、订单、聊天等数据 | 8.0 |
| 5 | Git | 拉取/管理项目代码 | 最新稳定版 |
| 6 | VS Code | 查看和修改代码、文档 | 最新稳定版 |

## 二、下载地址

- 微信开发者工具：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
- Node.js：https://nodejs.org/
- Python：https://www.python.org/downloads/windows/
- MySQL Community Server：https://dev.mysql.com/downloads/mysql/

- VS Code：https://code.visualstudio.com/

## 三、安装注意事项

### 1. Node.js

安装时一路下一步即可。安装完成后打开 PowerShell，输入：

```bash
node -v
npm -v
```

能显示版本号就表示安装成功。

### 2. Python

安装 Python 时必须勾选：

```text
Add python.exe to PATH
```

安装完成后打开 PowerShell，输入：

```bash
python --version
pip --version
```

能显示版本号就表示安装成功。

### 3. MySQL

安装 MySQL 时要记住自己设置的 root 密码，后面配置 `backend/.env` 会用到。

建议设置一个自己能记住的密码，例如：

```text
123456
```

如果是正式环境，不要使用简单密码。

安装完成后打开 PowerShell，输入：

```bash
mysql --version
```

能显示版本号就表示安装成功。如果提示找不到 `mysql`，说明 MySQL 没有加入环境变量，可以用 MySQL Installer 修复，或重新安装时勾选配置环境变量。

### 4. 微信开发者工具

安装后需要扫码登录。首次打开项目时，建议打开：

```text
miniapp/dist
```

如果打开项目根目录，也可以，但不要随意改 `project.config.json`。

## 四、安装完成检查

打开 PowerShell，依次输入：

```bash
node -v
npm -v
python --version
pip --version
mysql --version
git --version
```

每一条都能显示版本号，才算电脑基础环境准备完成。

## 五、常见安装问题

### PowerShell 提示命令不存在

通常是软件没有加入环境变量。优先重启电脑再试；如果仍然不行，重新安装对应软件，并注意勾选 `Add to PATH`。

### npm install 很慢或失败

可以切换 npm 国内镜像：

```bash
npm config set registry https://registry.npmmirror.com
```

然后重新执行：

```bash
npm install
```

### pip install 很慢或失败

可以临时使用国内镜像：

```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### 微信开发者工具提示域名不合法

开发阶段打开：

```text
详情 -> 本地设置 -> 不校验合法域名、web-view、TLS 版本以及 HTTPS 证书
```

正式上线时需要配置 HTTPS 域名。
