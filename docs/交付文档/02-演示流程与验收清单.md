# 演示流程与验收清单

本文档用于答辩、验收、给老师或同学演示项目。按顺序演示，能覆盖项目主要功能。

## 一、演示前准备

先确认四个窗口都已经启动：

| 窗口 | 目录 | 命令 | 作用 |
|---|---|---|---|
| 1 | `backend/` | `.\venv\Scripts\python.exe run.py` | 后端 HTTP 接口 |
| 2 | `backend/` | `.\venv\Scripts\python.exe ws_server.py` | 实时聊天服务 |
| 3 | `miniapp/` | `npm run dev:weapp` | 小程序编译 |
| 4 | `admin-web/` | `npm run dev` | 后台管理系统 |

同时确认微信开发者工具已经打开：

```text
miniapp/dist
```

后台管理地址：

```text
http://127.0.0.1:5174
```

管理员账号：

```text
账号：admin
密码：admin123456
```

## 二、推荐演示路线

### 1. 用户登录

演示目标：证明小程序可以进入个人中心并使用用户身份。

操作：

1. 打开小程序
2. 进入“我的”
3. 点击登录
4. 查看昵称、信用分、我的发布、我的订单等入口

验收：

- [ ] 能进入首页
- [ ] 能进入“我的”页面
- [ ] 用户信息能正常显示

### 2. 发布商品

演示目标：证明卖家可以发布二手商品。

操作：

1. 点击底部中间发布按钮
2. 填写标题、价格、分类、描述
3. 上传图片
4. 选择交易地点
5. 点击发布

建议演示商品：

```text
标题：二手计算机网络教材
价格：20
分类：书籍教材
描述：课程结束后闲置，书页完整，适合期末复习。
地点：图书馆门口
```

验收：

- [ ] 商品可以提交
- [ ] 商品能在首页或我的发布中看到
- [ ] 图片能正常显示
- [ ] 地点/距离信息能显示

### 3. 附近商品浏览和搜索

演示目标：证明 LBS 附近检索和商品浏览可用。

操作：

1. 回到首页
2. 查看商品列表
3. 切换分类
4. 使用关键词搜索
5. 进入商品详情
6. 查看距离、描述、卖家信息

验收：

- [ ] 首页能显示商品列表
- [ ] 商品详情能打开
- [ ] 搜索或分类筛选可用
- [ ] 距离信息或位置相关信息可见

### 4. 收藏与浏览历史

演示目标：证明用户行为记录功能可用。

操作：

1. 在商品详情页点击收藏
2. 进入“我的收藏”查看
3. 浏览几个商品
4. 进入“浏览历史”查看

验收：

- [ ] 商品可以收藏
- [ ] 收藏列表可以看到商品
- [ ] 浏览历史可以记录商品

### 5. 聊天和意向订单

演示目标：证明买家可以联系卖家，并自动形成交易意向。

操作：

1. 在商品详情页点击“联系卖家”
2. 进入聊天页
3. 发送一条消息，例如“你好，这个还在吗？”
4. 返回“我的消息”查看会话
5. 进入“我的订单”查看意向订单

验收：

- [ ] 聊天页能打开
- [ ] 消息可以发送
- [ ] 会话列表有记录
- [ ] 我的订单中出现对应订单

### 6. 完成订单和评价

演示目标：证明交易闭环完整。

操作：

1. 进入“我的订单”
2. 找到进行中的订单
3. 点击确认完成
4. 进入评价页
5. 选择星级并填写评价

建议评价：

```text
交易顺利，商品描述真实，沟通及时。
```

验收：

- [ ] 订单状态可以变成已完成
- [ ] 商品状态能变为已售或下架
- [ ] 可以提交评价
- [ ] 信用分或评价记录能更新

### 7. 后台管理系统

演示目标：证明管理员可以治理平台。

操作：

1. 打开后台地址 `http://127.0.0.1:5174`
2. 使用管理员账号登录
3. 查看工作台数据
4. 进入商品管理
5. 查看用户管理、订单管理、评价管理、举报处理、操作日志

验收：

- [ ] 后台可以登录
- [ ] 工作台数据能显示
- [ ] 商品列表能显示
- [ ] 用户列表能显示
- [ ] 订单、评价、举报、日志页面能打开

## 四、演示时常见问题

### 小程序请求失败

优先检查：

1. `.\venv\Scripts\python.exe run.py` 是否还在运行
2. `miniapp/.env` 里的 IP 是否是当前电脑 IP
3. 微信开发者工具是否勾选“不校验合法域名”
4. Windows 防火墙是否拦截 5002 端口

### 聊天消息发不出去

优先检查：

1. `.\venv\Scripts\python.exe ws_server.py` 是否还在运行
2. `miniapp/.env` 里的 `TARO_APP_WS` 是否使用当前电脑 IP
3. 5001 端口是否被防火墙拦截

### 后台登录失败

优先检查：

1. 后端 HTTP 服务是否运行
2. 数据库迁移是否执行
3. 管理员账号是否为 `admin`
4. 管理员密码是否为 `admin123456`

### 图片看不见

优先检查：

1. 后端服务是否运行
2. 上传目录 `backend/uploads` 是否存在
3. 小程序配置的 API 地址是否正确

### 数据库连接失败

优先检查：

1. MySQL 服务是否启动
2. `backend/.env` 里的数据库密码是否正确
3. 数据库 `lbs_secondhand` 是否已经创建
4. 是否执行过 `.\venv\Scripts\python.exe -m flask db upgrade`
