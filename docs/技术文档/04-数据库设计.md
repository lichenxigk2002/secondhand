# 数据库表结构设计

**版本：** V1.0  
**说明：** 15 张表，覆盖用户管理、商品、交易、通讯、评价、举报及管理后台  

**迁移方式：** 使用 Flask-Migrate (Alembic)，迁移脚本位于 `backend/migrations/versions/`  
执行：`flask db upgrade` 或 `alembic upgrade head`  

---

## 一、用户与权限

### 1. user（用户表）

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | INT | PK, AUTO_INCREMENT | 主键 |
| openid | VARCHAR(64) | UNIQUE, NOT NULL | 微信 openid |
| nick_name | VARCHAR(64) | DEFAULT '' | 昵称 |
| avatar | VARCHAR(512) | DEFAULT '' | 头像 URL |
| phone | VARCHAR(20) | DEFAULT '' | 手机号 |
| campus | VARCHAR(64) | DEFAULT '' | 校内区域/楼栋 |
| credit_score | INT | DEFAULT 100 | 信用分 |
| status | TINYINT | DEFAULT 1 | 状态：0 禁用 1 正常 |
| create_time | DATETIME | DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| update_time | DATETIME | ON UPDATE CURRENT_TIMESTAMP | 更新时间 |

### 2. admin（管理员表）

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | INT | PK, AUTO_INCREMENT | 主键 |
| username | VARCHAR(32) | UNIQUE, NOT NULL | 登录账号 |
| password | VARCHAR(255) | NOT NULL | 密码（加密存储） |
| real_name | VARCHAR(32) | DEFAULT '' | 真实姓名 |
| role | VARCHAR(16) | DEFAULT 'admin' | 角色：admin/super_admin |
| status | TINYINT | DEFAULT 1 | 状态：0 禁用 1 正常 |
| last_login_time | DATETIME | NULL | 最后登录时间 |
| create_time | DATETIME | DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| update_time | DATETIME | ON UPDATE CURRENT_TIMESTAMP | 更新时间 |

## 二、商品

### 3. category（商品分类表）

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | INT | PK, AUTO_INCREMENT | 主键 |
| parent_id | INT | DEFAULT 0 | 父级 ID，0 表示一级分类 |
| name | VARCHAR(32) | NOT NULL | 分类名称 |
| sort_order | INT | DEFAULT 0 | 排序值，越小越靠前 |
| status | TINYINT | DEFAULT 1 | 状态：0 禁用 1 启用 |
| create_time | DATETIME | DEFAULT CURRENT_TIMESTAMP | 创建时间 |

### 4. goods（商品表）

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | INT | PK, AUTO_INCREMENT | 主键 |
| user_id | INT | FK(user.id), NOT NULL | 发布用户 ID |
| category_id | INT | FK(category.id), DEFAULT 0 | 分类 ID |
| title | VARCHAR(128) | NOT NULL | 标题 |
| description | TEXT | NULL | 描述 |
| price | DECIMAL(10,2) | NOT NULL | 价格 |
| images | JSON | NULL | 图片 URL 数组 ["url1","url2"] |
| lat | DECIMAL(10,7) | NOT NULL | 纬度 |
| lng | DECIMAL(10,7) | NOT NULL | 经度 |
| address | VARCHAR(128) | DEFAULT '' | 位置描述 |
| status | TINYINT | DEFAULT 1 | 状态：0 下架 1 上架 2 已售 |
| audit_status | TINYINT | DEFAULT 0 | 审核：0 待审核 1 通过 2 驳回 |
| view_count | INT | DEFAULT 0 | 浏览次数 |
| create_time | DATETIME | DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| update_time | DATETIME | ON UPDATE CURRENT_TIMESTAMP | 更新时间 |

## 三、交易

### 5. orders（订单表）

*注：表名使用 orders 避免 MySQL 保留字 order 冲突*

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | INT | PK, AUTO_INCREMENT | 主键 |
| order_no | VARCHAR(32) | UNIQUE, NOT NULL | 订单号 |
| buyer_id | INT | FK(user.id), NOT NULL | 买家 ID |
| seller_id | INT | FK(user.id), NOT NULL | 卖家 ID |
| goods_id | INT | FK(goods.id), NOT NULL | 商品 ID |
| amount | DECIMAL(10,2) | NOT NULL | 交易金额 |
| status | TINYINT | DEFAULT 0 | 状态：0 待确认 1 进行中 2 已完成 3 已取消 |
| pay_type | VARCHAR(16) | DEFAULT '' | 支付方式：cash/online |
| remark | VARCHAR(256) | DEFAULT '' | 备注 |
| create_time | DATETIME | DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| complete_time | DATETIME | NULL | 完成时间 |

### 6. order_status_log（订单状态日志表）

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | INT | PK, AUTO_INCREMENT | 主键 |
| order_id | INT | FK(order.id), NOT NULL | 订单 ID |
| from_status | TINYINT | NULL | 原状态 |
| to_status | TINYINT | NOT NULL | 新状态 |
| operator_type | VARCHAR(16) | DEFAULT 'user' | 操作者类型：user/admin/system |
| operator_id | INT | DEFAULT 0 | 操作者 ID |
| remark | VARCHAR(256) | DEFAULT '' | 备注 |
| create_time | DATETIME | DEFAULT CURRENT_TIMESTAMP | 创建时间 |

---

## 四、即时通讯

### 7. conversation（会话表）

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | INT | PK, AUTO_INCREMENT | 主键 |
| user1_id | INT | FK(user.id), NOT NULL | 用户 1 ID（较小者） |
| user2_id | INT | FK(user.id), NOT NULL | 用户 2 ID（较大者） |
| goods_id | INT | FK(goods.id), DEFAULT 0 | 关联商品 ID，0 表示无关商品 |
| last_message_id | INT | DEFAULT 0 | 最后一条消息 ID |
| last_message_at | DATETIME | NULL | 最后消息时间 |
| user1_unread | INT | DEFAULT 0 | 用户 1 未读数 |
| user2_unread | INT | DEFAULT 0 | 用户 2 未读数 |
| create_time | DATETIME | DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| update_time | DATETIME | ON UPDATE CURRENT_TIMESTAMP | 更新时间 |

### 8. message（消息表）

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | INT | PK, AUTO_INCREMENT | 主键 |
| conversation_id | INT | FK(conversation.id), NOT NULL | 会话 ID |
| from_user_id | INT | FK(user.id), NOT NULL | 发送者 ID |
| to_user_id | INT | FK(user.id), NOT NULL | 接收者 ID |
| content | TEXT | NOT NULL | 消息内容 |
| msg_type | VARCHAR(16) | DEFAULT 'text' | 类型：text/image |
| extra | JSON | NULL | 扩展（如图片 URL） |
| is_read | TINYINT | DEFAULT 0 | 是否已读：0 未读 1 已读 |
| create_time | DATETIME | DEFAULT CURRENT_TIMESTAMP | 创建时间 |

### 9. message_read（消息已读记录表）

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | INT | PK, AUTO_INCREMENT | 主键 |
| message_id | INT | FK(message.id), NOT NULL | 消息 ID |
| user_id | INT | FK(user.id), NOT NULL | 读者 ID |
| read_time | DATETIME | DEFAULT CURRENT_TIMESTAMP | 阅读时间 |

## 五、评价与信用

### 10. evaluation（评价表）

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | INT | PK, AUTO_INCREMENT | 主键 |
| order_id | INT | FK(order.id), NOT NULL | 订单 ID |
| from_user_id | INT | FK(user.id), NOT NULL | 评价人 ID |
| to_user_id | INT | FK(user.id), NOT NULL | 被评价人 ID |
| role | VARCHAR(8) | NOT NULL | 评价角色：buyer/seller |
| star | TINYINT | NOT NULL | 星级 1-5 |
| comment | VARCHAR(512) | DEFAULT '' | 评语 |
| create_time | DATETIME | DEFAULT CURRENT_TIMESTAMP | 创建时间 |

### 11. credit_log（信用分变更记录表）

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | INT | PK, AUTO_INCREMENT | 主键 |
| user_id | INT | FK(user.id), NOT NULL | 用户 ID |
| change_value | INT | NOT NULL | 变更值，正为加负为减 |
| before_score | INT | NOT NULL | 变更前分数 |
| after_score | INT | NOT NULL | 变更后分数 |
| reason | VARCHAR(32) | DEFAULT '' | 原因：evaluation_good/evaluation_bad/report等 |
| ref_id | INT | DEFAULT 0 | 关联 ID（如评价 ID） |
| create_time | DATETIME | DEFAULT CURRENT_TIMESTAMP | 创建时间 |

### 12. report（举报表）

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | INT | PK, AUTO_INCREMENT | 主键 |
| reporter_id | INT | FK(user.id), NOT NULL | 举报人 ID |
| target_type | VARCHAR(16) | NOT NULL | 举报对象：goods/user/message |
| target_id | INT | NOT NULL | 举报对象 ID |
| reason | VARCHAR(32) | DEFAULT '' | 举报原因：fake/fraud/inappropriate等 |
| content | VARCHAR(512) | DEFAULT '' | 举报说明 |
| status | TINYINT | DEFAULT 0 | 处理状态：0 待处理 1 已处理 2 已驳回 |
| handler_id | INT | DEFAULT 0 | 处理人（管理员）ID |
| handle_remark | VARCHAR(256) | DEFAULT '' | 处理备注 |
| create_time | DATETIME | DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| handle_time | DATETIME | NULL | 处理时间 |

## 六、辅助功能

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|

| title | VARCHAR(64) | NOT NULL | 标题 |
| content | VARCHAR(512) | DEFAULT '' | 内容 |
| type | VARCHAR(16) | DEFAULT 'system' | 类型：system/order/message/report |
| ref_id | INT | DEFAULT 0 | 关联 ID |

## 八、索引建议

| 表名 | 索引 | 说明 |
|------|------|------|
| user | openid (UNIQUE) | 登录查询 |
| goods | user_id, status, category_id | 列表筛选 |
| goods | (lat, lng) | LBS 邻近检索，可配合空间索引 |
| goods | create_time | 时间排序 |
| order | buyer_id, seller_id, order_no | 订单查询 |
| message | conversation_id, create_time | 会话消息 |
| conversation | (user1_id, user2_id) UNIQUE | 会话唯一 |
| evaluation | order_id, to_user_id | 评价查询 |
| favorite | (user_id, goods_id) UNIQUE | 防重复收藏 |
| report | status, target_type | 举报列表 |
| operation_log | admin_id, create_time | 日志查询 |

---

## 九、ER 关系概要

```
user ──┬──< goods
       ├──< order (buyer/seller)
       ├──< conversation (user1/user2)
       ├──< message (from/to)
       ├──< evaluation (from/to)
       ├──< favorite
       ├──< report (reporter)
       └──< notification

goods ──< order
goods ──< conversation
category ──< goods

order ──< evaluation
order ──< order_status_log

conversation ──< message
message ──< message_read

admin ──< operation_log
admin ──< report (handler)
```
