# 论文插图 Word 重绘底稿

这份文档不是最终插图文件，而是给你在 Word 中重绘用的“临摹底稿”。

统一建议：

1. 全部使用白底、黑字、黑灰边框。
2. 框全部用矩形或圆角矩形，不用阴影、不用渐变。
3. 连线统一用直线或直角折线，不用曲线。
4. 标题统一放在图下方，例如“图 3-1 系统总体架构图”。
5. 字体建议宋体或黑体，字号以 Word 默认 10.5 磅到 12 磅为主。
6. 同一张图中的模块大小尽量一致，层次关系靠位置表达，不靠颜色表达。

└──────────────────────────────────────────────────────────────┘
                              │

│                           业务层                             │

└──────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌──────────────────────────────────────────────────────────────┐
│                        数据访问层                            │
│                  ┌────────────────────┐                      │
│                  │ SQLAlchemy ORM     │                      │
│                  └────────────────────┘                      │
└──────────────────────────────────────────────────────────────┘

Word 重绘要点：

1. 用 4 个大矩形表示 4 层。
2. 每层内部的小模块左右对齐。
3. 层与层之间只保留一条竖向箭头，结构会很稳。

## 图 3-5 数据库 ER 图

这张图不要把所有表都画上去，只画核心表，不然会很乱。建议只画 8 个。

```text
User ─────< Goods >───── Category
  │            │
  │            │
  │            ▼
  │          Orders
  │         /      \
  │        /        \
  ▼       ▼          ▼
Conversation       Evaluation
  │                  │
  ▼                  ▼
Message            CreditLog

User ─────< Favorite >──── Goods
User ─────< BrowseHistory > Goods
User ─────< Report
```

建议你在 Word 里实际画成：

1. `User`
2. `Goods`
3. `Category`
4. `Orders`
5. `Conversation`
6. `Message`
7. `Evaluation`
8. `Report`

连接关系建议：

1. User 与 Goods：一对多
2. Category 与 Goods：一对多
3. Goods 与 Orders：一对多
4. User 与 Orders：买家/卖家双关联
5. Conversation 与 Message：一对多
6. Orders 与 Evaluation：一对多
7. User 与 Report：一对多

Word 重绘要点：

1. 表名写在矩形上半部分。
2. 主键只保留 `id`，其他字段每表写 2 到 4 个代表字段即可。
3. 不要把 15 张表一次性全画进去。

## 图 4-4 聊天模块时序图

这张图建议用 5 个竖向对象。

```text
用户A        小程序端        Flask API      WebSocket服务      用户B
 │              │               │               │              │
 │ 发起聊天      │               │               │              │
 │────────────► │               │               │              │
 │              │ 获取/创建会话  │               │              │
 │              │──────────────►│               │              │
 │              │◄──────────────│ 返回会话信息   │              │
 │              │ 建立 WS 连接   │               │              │
 │              │──────────────────────────────►│              │
 │ 发送消息      │               │               │              │
 │────────────► │               │               │              │
 │              │ 发送消息       │               │              │
 │              │──────────────────────────────►│              │
 │              │               │ 写入数据库      │              │
 │              │               │◄──────────────►│              │
 │              │               │               │ 推送消息      │
 │              │◄────────────────────────────────────────────►│
```

Word 重绘要点：

1. 上方 5 个对象名横向排开。
2. 每个对象下方一条竖向虚线。
3. 消息传递全部用水平直线箭头。
